## 🌍 Overview / Descripción General

**(EN)** A high-conversion, ultra-fast landing page built for Akdemia EdTech (which serves 500+ schools). It is fully self-contained in a single 213KB HTML file for maximum performance. It connects natively to **Google Sheets via Google Apps Script** to capture leads in real-time and trigger automated email notifications.

**(ES)** Una landing page de altísimo rendimiento diseñada para Akdemia EdTech (más de 500 colegios). Está autocontenida en un único archivo HTML de 213KB para carga instantánea. Se conecta nativamente a **Google Sheets mediante Apps Script** para guardar leads en tiempo real y disparar notificaciones automáticas por correo.
