## 🎯 What is This?

A **high-performance lead capture landing page** built entirely in vanilla HTML/CSS/JavaScript for **Akdemia**, a Venezuelan EdTech company. Integrates directly with Google Sheets via Apps Script for real-time lead management — zero backend server needed.

---

## ⚡ Core Features

- 📋 **Lead capture form** — collects prospect info for sales pipeline
- 📊 **Google Sheets integration** — submissions saved in real-time via Apps Script
- 📧 **Automated email alerts** — triggers notifications on each new lead
- 🎨 **Self-contained** — single HTML file, all CSS and JS embedded (213KB)
- 📱 **Fully responsive** — all screen sizes
- ⚡ **Zero dependencies** — no npm, no frameworks, no build step

---

## 🏗️ How It Works

```
User fills form → JS validates → POST to Apps Script webhook
                                          ↓
                                 Saves row to Google Sheet
                                          ↓
                                 Sends email notification
                                          ↓
                                 Sales team follows up
```

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Pure HTML5, CSS3, Vanilla JavaScript |
| Backend | Google Apps Script (serverless) |
| Database | Google Sheets (real-time) |
| Notifications | Gmail API via Apps Script |
| Hosting | Any static host |
